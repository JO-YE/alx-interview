# 0x05. N Queens

**Algorithm | Python**

## Description

The N Queens puzzle is the challenge of placing N non-attacking queens on an NÃ—N chessboard. This project provides a Python program that solves the N Queens problem for a given N.

- The program ``0-nqueens.py` is a Python script that solves the N Queens problem using a backtracking algorithm. It checks all possible combinations of queen placements on the chessboard and outputs the solutions.
